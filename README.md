
🤖 GenAI Project

A Generative AI powered application that creates intelligent responses, content, or insights based on user input. 
This project integrates modern AI models with a scalable web architecture to deliver fast and meaningful results.



🚀 Features

✨ AI-powered content generation
💬 Interactive user interface with smooth animations using Framer Motion
⚡ Fast API responses
🔐 Secure authentication 
💳 Online payments and subscriptions powered by Stripe
📊 Scalable backend architecture
🌐 Real-time or on-demand generation
📁 History or session tracking

🛠 Tech Stack:
 -----Frontend -----
      React.js,
      Tailwind CSS,
      Framer Motion for modern UI animations and transitions

------Backend-----
     Node.js,
     Express.js,
     Payment processing and billing integration using Stripe

-----Database-------
     MongoDB 

-----AI Integration-----
     Model access and routing via OpenRouter,
     Advanced language reasoning and generation using DeepSeek models,
     OpenAI API 

-----Other Tools-----
  Git & GitHub,
  Environment Variables
